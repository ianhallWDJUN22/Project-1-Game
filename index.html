BOMB ARMED
--:--

*WARNING*

If the red light is on this bomb is armed. Only the disarm code will prevent detonation. As a reference, the system will flash each step of the disarm code in the correct order. After each step you must correctly repeat the code exactly as it was displayed. Incorrect inputs will reset the disarm process. Failure to input all steps of the code before the timer reaches 00:00 will result in detonation. Remove panel to begin disarm process.
